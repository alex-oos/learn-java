# springboot-demo

#### 介绍

springboot-demo项目

#### 软件架构

springboot+mybatis-plus+swagger+jwt

#### 功能

1. 登录
   > jwt令牌，使用的hutools工具
2. 拦截器
   > 拦截未登录的接口
3. yml文件读取
   1. @value和@PropertySource 进行读取
   2. @ConfigurationProperties 和 @PropertySource 进行读取
4. swagger集成
   > http://localhost:8080/swagger-ui/index.html
